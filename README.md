# Arul Nigam - Portfolio

A high-end, minimalist, one-page personal portfolio with a "Brutalist Editorial" aesthetic.

## Features

- **Brutalist Editorial Design**: Sophisticated spacing and bold typography
- **Paper Texture**: Natural warm paper background with subtle noise overlay
- **Smooth Animations**: Framer Motion scroll-reveal animations
- **Responsive Design**: Optimized for all screen sizes
- **Floating Navigation**: Centered dock with smooth scroll navigation

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
